# Memory Game Project

[1] How to Play
[2] Credit
[3] Issues

-----

1) How to Play

  - open index.html
  - to play, click on any cards
  - to replay or reset, click on restart button
  - reloading the page or closing the modal will start a new game

2) Credit

3) Issues
